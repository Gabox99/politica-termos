Política de Privacidade

Levamos sua privacidade a sério. Esta política descreve como tratamos os dados dos usuários do nosso serviço.

1. Coleta de Dados

Não coletamos, armazenamos ou compartilhamos dados pessoais de nossos usuários. Nosso serviço é baseado exclusivamente na interação com a API do TikTok.

2. Compartilhamento de Dados

Não compartilhamos nenhuma informação com terceiros, pois não coletamos dados de nossos usuários.

3. Uso de Dados

Qualquer dado eventualmente processado pela API do TikTok é utilizado apenas para a funcionalidade do serviço e não é armazenado por nós.

4. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. As alterações serão publicadas nesta página.

5. Contato

Para mais informações, entre em contato conosco pelo e-mail: tiktokcortfutebol@hotmail.com.
