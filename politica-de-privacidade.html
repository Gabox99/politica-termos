Política de Privacidade - Cortes Futebolísticos

Valorizamos sua privacidade. Aqui explicamos como lidamos com suas informações ao usar nosso serviço de cortes relacionados ao futebol.

1. Coleta de Dados

Não coletamos, armazenamos ou compartilhamos dados pessoais de nossos usuários. Utilizamos apenas a API do TikTok para exibir cortes de vídeos.

2. Compartilhamento de Dados

Não compartilhamos informações com terceiros, já que não coletamos dados pessoais.

3. Uso de Dados

Dados processados pela API do TikTok são utilizados exclusivamente para as funcionalidades do serviço e não são armazenados por nós.
